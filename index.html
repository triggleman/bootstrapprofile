What's Your Gym ?

What's Your Gym is a Ruby on Rails Full Stack web app search engine that allows users to search up any gym location based on their zipcode, city, or state.

GitHired

GitHired is a Ruby on Rails based web app that uses Github job API and allows us web developers to search for jobs in a certain areas and you can also specify their desired language.

iScream4IceCream

iScream4IceCream is a React-Native mobile app that allows a user to sign-up, login, search, and save to favorites. iScream4IceCream uses Yelp API and Ngrok for running the app. iScream allows users to be in any location and search for a ice cream place and see information about it. You can save it to your favorites to remind the user to tell their friends about great ice cream experience they had.

Splat!

Splat is a Javascript game that allows two players to battle each other. The objective of the game is to splat Donald Trump’s face with your color and connect 4 colors in a row. The first person to splat Donald Trump’s face 4 in a row with your color wins!
